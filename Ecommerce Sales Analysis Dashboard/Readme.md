# 📊 Ecommerce Sales Dashboard (Excel Project)

## 📌 Project Overview
This project is an **end-to-end Excel dashboard** built to analyze e-commerce sales data.  
It provides insights into sales performance, customer behavior, and product trends using interactive visualizations.

---

## 🎯 Objective
- Analyze sales data to identify key trends
- Track revenue, profit, and order metrics
- Build an interactive dashboard for decision-making
- Improve Excel data analysis and visualization skills

---

## 🛠️ Tools & Technologies
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers & Filters
- Data Cleaning Techniques

---

## 📂 Dataset
The dataset contains:
- Order details
- Sales and profit information
- Customer and region data
- Product categories

---

## 📊 Key Features
- 📈 Sales Trend Analysis
- 💰 Profit & Revenue Insights
- 🌍 Region-wise Performance
- 📦 Category-wise Sales Breakdown
- 🎛️ Interactive Dashboard with Filters
